# Chrome Rectangle Open Links

Chrome 拡張機能 (Manifest V3)。ページ上で矩形選択した範囲内のリンク (`a[href]`) を抽出し、重複を除去して別タブで一括表示します。

## 構成

| ファイル | 役割 |
| --- | --- |
| `manifest.json` | MV3 マニフェスト。content script / service worker / action を定義 |
| `content.js` | 矩形選択モードを管理。ドラッグで矩形を描画し、範囲内のリンクを抽出・重複除去して service worker へ送信 |
| `background.js` | Service worker。アイコンクリックで content script へ開始メッセージを送り、受け取った URL を `chrome.tabs.create` で一括タブ展開 |

## 使い方

1. Chrome で `chrome://extensions` を開く
2. 右上の「デベロッパー モード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」でこのフォルダを選択
4. リンクを一括表示したいページを開く
5. ツールバーの拡張機能アイコンをクリックする（ポップアップは出ません）
6. ページ上でマウスドラッグし、リンクを含む矩形を描く
7. ドラッグを離すと、矩形内のリンク（重複除去済み・`http`/`https` のみ）が新しいタブで一括表示される

## 操作のヒント

- 選択中はカーソルが十字になります。上部に案内バーが表示されます。
- **Esc** キーで選択モードをキャンセルできます。
- リンクの中心点が矩形内に入ったものを抽出対象とします。
- `javascript:` / `mailto:` / `#anchor` など `http(s)` 以外のリンクは除外されます。
- `chrome://` ページや Chrome ウェブストアなど、拡張機能が動作しないページがあります。

## 動作確認済みの最小構成

本バージョンは最小構成です。アイコンクリックで開始 → ドラッグで矩形 → 即時全タブ展開、という流れで動作します。
